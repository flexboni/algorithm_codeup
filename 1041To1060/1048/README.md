# [CodeUp 기초 100제](https://codeup.kr/problem.php)

## [1048 : [기초-비트시프트연산] 한 번에 2의 거듭제곱 배로 출력하기(설명)](https://codeup.kr/problem.php?id=1048)

시간 제한: 1 Sec 메모리 제한: 128 MB

## 문제 설명

    정수 2개(a, b)를 입력받아 a를 2b배 곱한 값으로 출력해보자.
    0 <= a <= 10, 0 <= b <= 10

### 입력

    정수 2개가 공백을 두고 입력된다.
    0 <= a, b <= 10

### 출력

    a 를 2b배 만큼 곱한 값을 출력한다.

### 입력 예시

    1 3

### 출력 예시

    8
